    使用提醒

    • `Meta` 在 Linux 浏览器里通常就是 `Super / Win`。
    • 很多系统会把物理助手键映射成 `F23`，有些浏览器再把它命名成 `XF86Assistant`。
    • 如果你只看到 `Shift` 和 `Meta`，但一直没有主键事件，通常是桌面环境先把这个键吃掉了。
